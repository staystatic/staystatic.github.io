beer.db - New Repo /maps - Free 'Full-Screen' Interactive Beer Maps w/ Brewery Listings

Tue Nov 11 2014 01:00:00 GMT+0100 (W. Europe Standard Time)

The beer.db project - offering free public domain beer, brewery and brewpubs data - added a new repo, that is, /maps for hosting 'full-screen' interactive beer maps with brewery listings.

See an example beer map for Austria (~200 breweries n brewpubs) live or check the source using the mapbox.js mapping library.
